The Good Template

A responsive web site framework

What is The Good Template?

The Good Template is a responsive web design framework designed in LESS. It attempts to quickly lay out helpful text baselines, form styles, nifty features, and a simple fixed & responsive grid.

Features.

  • Awesome text baseline.
  • Stupendous responsive grid.
  • Grand form styling.
    1. Validation!
    2. Selects!
  • Strong HTML5 templates.
  • Crazy LESS mixins.
